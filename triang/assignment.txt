Тестовое задание

Написать программу, которая выполняет триангуляцию двухмерной области.
Область односвязная, ограниченная только одним контуром (без дырок).
Сам контур задаётся множеством точек по их координатам.
Точки должны являться узлами разбиений на контуре.

Работу надо оформить в виде функции, на входе которой вектор точек:

vector<Point> Contour, где  Point - масив из трех координат.
Третья координата обнулена.
На выходе - вектор треугольников vector<Triаngle> Mesh,
  где Triangle - массив из трех номеров точек,
  хранящихся в общем массиве точек vector<Point>AllPoints.
